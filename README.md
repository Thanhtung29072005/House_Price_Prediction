House Price Prediction (Linear Regression from Scratch)

Dự án Machine Learning dự đoán giá nhà dựa trên các đặc trưng như diện tích, số phòng ngủ, sử dụng Linear Regression tự cài đặt bằng NumPy (không dùng scikit-learn).

---

## Mục tiêu
- Hiểu rõ cách hoạt động của Linear Regression
- Tự cài đặt Gradient Descent
- Thực hành :
  - Load dữ liệu
  - Chuẩn hóa 
  - Chia train / test
  - Huấn luyện mô hình
  - Đánh giá bằng MSE
